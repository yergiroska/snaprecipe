# SnapRecipe 📸

Mobile AI app with computer vision for recipe generation.

Takes a photo of available ingredients, uses Groq Vision (Llama 4 Scout) to detect them, and generates personalized recipes. Users can save favorite recipes and filter by preparation time or diet type.

## Stack
- React Native + Expo (TypeScript)
- Expo Camera
- Groq Vision API
- Firebase Auth + Firestore
- React Navigation
